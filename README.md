# Unnamed Operating System
## 何これ？
これは、OSを作るプロジェクトです。
現在、実装言語はNetwide Assemblyを使っています。

## どう使うの？
```bash
$ git clone https://github.com/pixie-grasper/operating-system.git
$ cd operating-system
$ make
```
する事によって、ISOイメージが出来上がります。

## 何ができるの？
まだ何もできません。
